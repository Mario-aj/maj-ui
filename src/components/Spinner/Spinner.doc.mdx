# Spinner

The component to manage te loading state and so on.

## Props

| Prop   | Description          | Type                                        | Default   |
| ------ | -------------------- | ------------------------------------------- | --------- |
| intent | The color of Spinner | 'primary', 'success', 'secondary', 'danger' | 'primary' |
| size   | The size of Spinner  | 'sm', 'md', 'lg'                            | 'md'      |

## Usage

> Note: considering that this library has not yet been published, the path of import is fictitious

```jsx
import { Spinner } from '@maj/react-ui';

const LoadingState = (): JSX.Element => <Spinner size="md" intent="danger" />;
```
